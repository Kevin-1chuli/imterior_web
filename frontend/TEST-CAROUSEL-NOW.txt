⚠️ IF SOMETHING'S WRONG
───────────────────────────────────────────────────────────────────────────

Problem: "Nothing shows up"
  
  → Press F12 → Console tab
  → Look for red error messages
  → Common errors:
     ❌ "Failed to load resource: 404" = File path wrong
     ❌ "hero__media not found" = HTML structure issue
  
  → If 404 error, verify file structure:
     frontend/ngb.html  ← Your HTML file
     ../assets/images/home/download (1).jpg  ← Must exist
     ../assets/images/home/image0.jpg        ← Must exist
     ../assets/images/home/image3 (1).jpg    ← Must exist
     ../assets/images/home/image4 (1).jpg    ← Must exist
     ../assets/videos/0620(1).mp4            ← Must exist

───────────────────────────────────────────────────────────────────────────

Problem: "Images show but no zoom effect"
  
  → This is a CSS issue
  → Check: frontend/styles/ngb.css
  → Look for: @keyframes kenBurnsZoom
  → Should contain: transform: scale(1.08);

───────────────────────────────────────────────────────────────────────────

Problem: "Video won't play"
  
  → This is NORMAL on some browsers (autoplay policy)
  → Console shows: "Video autoplay blocked"
  → Carousel continues with images only (graceful fallback)
  → This is expected behavior, no fix needed!

───────────────────────────────────────────────────────────────────────────

Problem: "Text not readable"
  
  → Open: frontend/styles/ngb.css
  → Find line ~709: background: linear-gradient(...)
  → Change: rgba(10, 8, 5, 0.4) to rgba(10, 8, 5, 0.6)
  → This makes overlay darker

🔄 QUICK CUSTOMIZATIONS

Change how long each image displays:
  → File: frontend/scripts/ngb.js
  → Line: ~58

  → File: frontend/styles/ngb.css
