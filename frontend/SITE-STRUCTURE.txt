🔄 USER INTERACTION FLOWS
═══════════════════════════════════════════════════════════════════════════


FLOW 1: Quick Preview (Lightbox)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   User on Homepage
         │
         ├─→ Scrolls to Furniture Showcase
         │
         ├─→ Clicks on FURNITURE IMAGE
         │
         ├─→ Lightbox Opens (overlay)
         │       │
         │       ├── Shows large image
         │       ├── Product title
         │       ├── Category
         │       └── [View Full Details] button → furniture.html
         │
         └─→ Clicks X or ESC → Closes lightbox, stays on homepage


FLOW 2: Full Product Details
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   User on Homepage
         │
         ├─→ Scrolls to Furniture Showcase
         │
         ├─→ Clicks "VIEW DETAILS →" button
         │
         ├─→ Navigates to furniture.html
         │       │
         │       ├── Views full product info
         │       ├── Reads detailed description
         │       ├── Sees price range
         │       ├── Fills customization form
         │       │     ├── Selects color
         │       │     ├── Selects material
         │       │     ├── Selects size
         │       │     ├── Selects finish
         │       │     └── Adds special requests
         │       │
         │       └── Clicks "Request Quote"
         │               │
         │               └─→ Alert confirmation
         │                   └─→ Form resets
         │
         └─→ Clicks "← Back to Home" → Returns to ngb.html


FLOW 3: Category Filtering (Homepage)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   User on Homepage
         │
         ├─→ Scrolls to Furniture Showcase
         │
         ├─→ Clicks filter button:
         │       ├── "All Furniture" → Shows all 4 items
         │       ├── "Sofas" → Shows 1 item (sofa)
         │       ├── "Dining" → Shows 1 item (dining set)
         │       ├── "Wardrobes" → Shows 1 item (wardrobe)
         │       └── "Coffee Tables" → Shows 1 item (coffee table)
         │
         └─→ Gallery updates instantly (no page reload)

📊 DATA MANAGEMENT
═══════════════════════════════════════════════════════════════════════════

WHERE FURNITURE DATA IS STORED:

ngb.js (line ~950)
    │
    └─→ furnitureData array
            │
            ├─→ Product 1: Custom Luxury Sofa
            │     ├── id, title, category
            │     ├── image path
            │     ├── descriptions (short & detailed)
            │     ├── material
            │     ├── priceRange (min, max)
            │     └── customization options
            │           ├── colors []
            │           ├── materials []
            │           ├── sizes []
            │           └── finishes []
            │
            ├─→ Product 2: Built-in Wardrobe
            ├─→ Product 3: Elegant Dining Set
            └─→ Product 4: Designer Coffee Table


WHERE DATA IS USED:

1. Homepage Gallery (ngb.html)
   └─→ Shows card with: image, title, short description

2. Lightbox Preview (ngb.html)
   └─→ Shows: large image, title, category

3. Product Detail Page (furniture.html)
   └─→ Currently hardcoded (not reading from furnitureData)
   └─→ Shows: all details, price, materials, customization form

🎨 STYLING SYSTEM
═══════════════════════════════════════════════════════════════════════════

Design Tokens (ngb.css):
    │
    ├─→ Colors
    │   ├── Black/Charcoal: Backgrounds, text
    │   ├── Gold: Accents, prices, CTAs
    │   ├── Cream/Parchment: Warmth, cards
    │   └── White: Contrast
    │
    ├─→ Typography
    │   ├── Playfair Display: Headlines, hero
    │   ├── Montserrat: Section headings, labels
    │   └── Poppins: Body text, descriptions
    │
    ├─→ Spacing (8-point grid)
    │   └── sp-1 through sp-16
    │
    └─→ Components
        ├── .btn (primary, secondary, outline)
        ├── .section (standard section container)
        └── Form elements


Page-Specific Styles:

furniture.css
    └─→ Gallery cards, filters, lightbox

furniture.html <style> tag
    └─→ Product page layout, sidebar, form
