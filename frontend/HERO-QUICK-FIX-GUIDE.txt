🛠️ TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────

Problem: "Nothing shows up"
→ Check browser console (F12) for error messages
→ Verify file paths match your folder structure
→ Make sure ngb.html is in frontend/ folder

Problem: "Video won't play"
→ This is normal in some browsers (autoplay policy)
→ Image carousel activates automatically as fallback
→ No action needed

Problem: "Images show but video doesn't"
→ Check video file exists at: assets/videos/0620(1).mp4
→ Check video format is .mp4 (H.264 codec)
→ Try opening video directly in browser to test

Problem: "Text not readable"
→ Overlay might be too light
→ Increase overlay opacity in CSS (see customizations above)
